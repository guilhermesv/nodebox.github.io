Creatures

In september 2006, we wrote a proposal for VAF, the Flemish Audiovisual Fund. VAF granted us fundings to develop a (small) experimental computer game project (workname "Creatures"). The game world will deal with plants and insects powered by a genetic algorithm.

What is Creatures?

Creatures is an experimental project bordering between art and gaming. It takes a lot of inspiration from the Evolution project in the NodeBox gallery. Insects will swarm around, plants will grow, life will virally adapt. As a player it will be your goal to keep life in balance. So gaming never stops. Our main partner in this project is Champ d'Action, a contemporary music ensemble. They will take care of the game audio and music according to a similar viral principle which we plan to use in the game world.

Open source software & artwork

We have released the game development engine: a free, cross-platform library for   generating 2D   animations with Python programming code. It is built on   Pyglet and   adopts the drawing API from NodeBox for Mac OS X.   It has built-in support for paths, layers,  motion tweening,   hardware-accelerated image effects, simple physics  and interactivity.
