Hi Martin,

Unfortunately our machines still have the ATI graphics card, so I won't be able to test this until I find a machine with a GeForce installed.

My best guess would be that Mac OS X itself needs some updates. We've experienced some odd things in the past if you used Core Image on the initial OS X 10.4.0 with a newer graphics card - the problem went away once you updated to OS X 10.4.9.

Leopard might bring a solution, or else I'll have to find someone with a GeForce to run some tests.
