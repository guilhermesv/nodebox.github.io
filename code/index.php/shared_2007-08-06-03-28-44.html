Posted by Tom De Smedt on Aug 06, 2007

Hi Guy,

Probably not. Because then we would need an interface where you can select what key to assign to "run", and for consistency we would then need switchable combinations for all commands etc. At this point it's too much work with too little gain.

So you'll have to choose "run" from the menu :-(

Best regards,

Posted by nex on Aug 06, 2007

If you can select it from the menu, you can trigger it through external tools, e.g. AppleScripts, Quicksilver itself, or some tool that lets you assign custom hotkeys to menu entries.

However, I would suggest changing that Quicksilver hotkey of yours -- several other applications use cmd-r as well and you're obscuring these, too.


Posted by anonymous on Aug 18, 2007

Actually, I'm not sure it will work with NB, but in the mouse & keyboard preference pane, under Keyboard Shortcuts, press the plus (+), select the app with the command, type in the exact name of the menu item you would like to reassign, and choose a shortcut for your liking. The menu command will appear under App Keyboard Shortcuts, All apps. The computer I am writing this on has 10.3, so I can't try it with NB. Best of luck!
