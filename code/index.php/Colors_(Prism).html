Colors (Prism)

Description

The Prism Colors library defines HSB color palettes for any given subject. Each color value is defined as a hue with certain shades, like dark blue, or intense blue. A color palette is retrieved using the Google library or the Yahoo library for NodeBox. A more extensive text on the how and why of the algorithm is here.
