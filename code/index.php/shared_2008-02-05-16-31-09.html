NodeBox 1.9.3

Posted by fdb on Feb 05, 2008



Hi,

We just released NodeBox 1.9.3. This release contains many requested features, some specifically created for the workshop in Helsinki.

Here's an overview of the new features:

  • Zooming - you can now zoom the canvas, even while an animation is running.
  • Boolean operations - make new paths by combining existing ones, like Adobe Illustrator's pathfinder.
  • Jumping to a line number now actually works!
  • Math - includes both Numeric and NumPy modules.
  • Specialization - includes Psyco to make graphs and cornu curves go faster.


The download is available from the homepage; and works with PowerPC and Intel on both Mac OS X 10.4 and 10.5.

Have fun!

Frederik
